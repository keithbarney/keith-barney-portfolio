# Design Style Guide

## Core Aesthetic

**Swiss Modernism meets Technical Systems**

Your design taste combines the precision of Swiss/International style with the functional beauty of information systems—cockpit displays, dot matrices, and data visualization.

---

## Typography

### Primary Characteristics
- Monospaced / technical fonts
- Dot-matrix / LED display letterforms
- Mixed weights: light labels, bold values
- Large numeric displays with decimal precision
- Extreme scale contrast (giant letters + small text)

### Style Details
- Vertical/rotated text as design element
- `//` code-style separators
- Type specimen aesthetics
- Engineered, not decorative

### Recommended Fonts
- **Monospace:** JetBrains Mono, IBM Plex Mono, Space Mono
- **Sans-serif:** Helvetica Neue, Inter, Suisse Int'l, Akkurat
- **Display:** GT America, Founders Grotesk, Monument Extended

---

## Color

### Palette Strategy
- **Foundation:** Black + white / warm neutrals (beige, taupe, warm gray)
- **Accent:** One bold color—typically red, orange, or amber
- **Contrast:** High contrast text on soft backgrounds

### Specific Values (Starting Points)
```
Background:  #F5F3EF (warm off-white)
             #1A1A1A (near-black)
Text:        #000000, #FFFFFF
Accent:      #E63B2E (red-orange)
             #FFB800 (amber/LED yellow)
Muted:       #949494 (secondary text)
```

## Layout & Grid

### Principles
- Rigid Swiss grid systems
- Asymmetric balance
- Single-column or modular compositions
- Visible grid lines as design elements
- Generous whitespace within structure

### Spacing
- Large margins as intentional design
- Dense information hierarchy
- Card-based UI with soft radius

---

## Visual Elements

### Shapes & Patterns
- Dot matrix / grid systems as texture
- Dithering patterns for gradients/depth
- Bold geometric forms (circles, rounded bars)
- Thin horizontal rules as dividers

### Data Visualization
- Abstract/artistic over literal
- Oversized chart elements as hero graphics
- Minimal axis lines and labels
- Radar charts, bar graphs, gauges

### Photography Integration
- B&W or desaturated
- Typography layering on images
- Editorial fashion aesthetic

---

## UI Components

### Cards
- Large rounded corners (16-24px radius)
- Soft shadows or no shadows
- Warm neutral backgrounds

### Navigation
- Minimal, spaced-out
- Monospaced metadata
- Vertical text labels where appropriate

### Data Display
- Cockpit/dashboard aesthetic
- Numeric precision (decimals, units)
- Small caps for labels
- Bold values

## Quick Reference Checklist

When designing, ask:

- [ ] Is typography the hero element?
- [ ] Is there extreme scale contrast?
- [ ] Limited to 2-3 colors max?
- [ ] Grid-based composition?
- [ ] Generous whitespace?
- [ ] Technical/systematic feeling?
- [ ] One bold accent color?
- [ ] Monospaced elements for data/metadata?
- [ ] Minimal decorative elements?
